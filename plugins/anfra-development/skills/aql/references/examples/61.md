```yaml
- field: id
  model: public_cities

  model: countries

  model: users

  model: product
measures:
- field: order_count
  model: ''
```

## Bad AQL
```aql
explore {
  dimensions {
    product_id: product.id,
    product_name: product.name,
  }
  measures {
    product_price: product.price,
  }
  sorts {
    product_price DESC,
    product_id ASC,
  }
}
```

## Why it's bad
* `product.price` is a **dimension** and must be put in `explore.dimensions` instead.

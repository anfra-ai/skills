## Dataset fields
```yaml
dimensions:
- field: sign_up_at
  model: users
- field: id
  model: users
- field: first_name
  model: users
- field: last_name
  model: users
- field: name
  model: city
measures:
- field: avg_age
  model: ''
- field: order_count
  model: users
```

## AQL
```aql
dimension users.full_name = concat(users.first_name, ' ', users.last_name);
metric full_name_and_order_count =
  concat(
    users | max(users.full_name),
    ' - ',
    cast(users.order_count, 'text'),
  )

explore {
  dimensions {
    id: users.id,
  }
  measures {
    full_name_and_order_count: full_name_and_order_count,
  }
  sorts {
    id ASC,
  }
}
```

## Learning points
* Dimensions (e.g. `users.full_name`) and metrics/measures (e.g. `users.order_count`) cannot be mixed directly in an expression. Need to aggregate or group the dimensions.

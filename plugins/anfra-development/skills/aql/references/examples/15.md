## Dataset fields
```yaml
dimensions:
- field: created_at
  model: db_users
measures:
- field: avg_age
  model: db_users
```

## AQL
```aql
explore {
  dimensions {
    user_created_at: db_users.created_at | month(), // NOTE: MUST apply date trunc
  }
  measures {
    avg_age_last_month: db_users.avg_age,
    avg_age_last_year_same_month: db_users.avg_age | relative_period(db_users.created_at, interval(-1 year)),
  }
  filters {
    db_users.created_at is @(last 3 months),
  }
  sorts {
    avg_age_last_month ASC,
  }
}
```

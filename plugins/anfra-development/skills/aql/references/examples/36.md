## AQL
```aql
metric order_count = orders | count(orders.id);
metric users_in_london = users | where(public_cities.name == 'London');
metric london_user_rank_by_order_count =
  users_in_london
  | rank(order: order_count | desc());
explore {
  dimensions {
    user_id: users.id,
    user_name: users.name,
    user_created_at: users.created_at,
  }
  measures {
    order_count: order_count,
    london_user_rank_by_order_count: london_user_rank_by_order_count,
  }
  filters {
    london_user_rank_by_order_count < 11,
  }
  sorts {
    order_count DESC,
    user_id ASC,
  }
}
```

## Learning points
* This question requires filtering **prior to** ranking.

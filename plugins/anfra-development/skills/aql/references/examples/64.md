## Dataset fields
```yaml
dimensions:
- field: id
  model: orders
- field: discount
  model: orders
- field: order_id
  model: items
- field: price
  model: items
measures: []
```

## Bad AQL
```aql
metric total_discount = orders | sum(orders.discount);
metric revenue =
  unique(orders.id)
  | select(value: items | sum(items.price) - total_discount)
  | sum(value);
explore {
  dimensions {
    id: orders.id,
  }
  measures {
    revenue: revenue,
  }
  sorts {
    id ASC,
  }
}
```

## Why it's bad
* `|` (pipe operator) has lower precedence than `-` (arithmetic operators). `items | sum(items.price) - total_discount` is equivalent to `items | (sum(items.price) - total_discount)`. Must parenthesize properly: `(items | sum(items.price)) - total_discount`.

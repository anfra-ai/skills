## Dataset fields
```yaml
dimensions:
- field: id
  model: public_cities
- field: name
  model: public_cities
- field: name
  model: countries
- field: id
  model: countries
- field: name
  model: users
- field: id
  model: users
- field: created_at
  model: users
- field: order_id
  model: items
- field: id
  model: orders
- field: id
  model: product
- field: name
  model: product
- field: price
  model: product
measures:
- field: order_count
  model: ''
```

## Bad AQL
```aql
explore {
  dimensions {
    city_id: public_cities.id,
    city_name: public_cities.name,
    country: countries.name,
  }
  measures {
    user_count: public_cities.user_count,
  }
  sorts {
    city_id ASC,
  }
}
```

## Why it's bad
* `public_cities.user_count` is a **dimension** and must be put in `explore.dimensions` instead.

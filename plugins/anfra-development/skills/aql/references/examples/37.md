## AQL
```aql
metric top_10_users =
  unique(users.id)
  | where(public_cities.name == 'London')
  | filter(rank(order: order_count | desc()) < 11);
explore {
  dimensions {
    user_id: users.id,
    user_name: users.name,
    user_created_at: users.created_at,
  }
  measures {
    order_count: order_count,
  }
  filters {
    users.id in top_10_users,
  }
  sorts {
    order_count DESC,
    user_id ASC,
  }
}
```

## Learning points
* This question requires filtering **prior to** ranking.
* Use existing metric `order_count` that is already available

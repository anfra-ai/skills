## Dataset fields
```yaml
dimensions:
- field: id
  model: orders
- field: discount
  model: orders
measures:
- field: value
  model: orders
```

## Bad AQL
```aql
metric revenue = orders.value - orders.discount;

explore {
  dimensions {
    id: orders.id,
  }
  measures {
    revenue: revenue,
  }
  sorts {
    id ASC,
  }
}
```

## Why it's bad
* `orders.discount` is a Dimension and cannot be calculated together with `orders.value`. How to fix: Must aggregate or group `orders.discount`.

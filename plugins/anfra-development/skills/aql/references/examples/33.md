## AQL

explore {
  dimensions {
    time: users.created_at | year(),
    city_id: cities.id,
    city_name: cities.name,
  }
  measures {
    total_user_value: running_total(users.value, users.created_at),
  }
  sorts {
    time ASC,
    city_id ASC,
  }
}

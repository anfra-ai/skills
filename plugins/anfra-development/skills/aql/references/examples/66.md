## Dataset fields
```yaml
dimensions:
- field: id
  model: public_cities
- field: name
  model: public_cities
- field: name
  model: countries
- field: name
  model: users
- field: id
  model: users
- field: created_at
  model: users
measures:
- field: order_count
  model: users
```

## Bad AQL
```aql
metric user_order_count = users | sum(users.order_count);
explore {
  dimensions {
    user_id: users.id,
    user_name: users.name,
    user_created_t: users.created_at,
  }
  measures {
    order_count: user_order_count,
  }
  sorts {
    user_id ASC,
  }
}
```

## Why it's bad
* `metric user_order_count = users | sum(users.order_count);` is **bad** because `users.order_count` is already aggregated. Should have written `metric user_order_count = users.order_count;`

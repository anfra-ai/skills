## AQL
```aql
dimension products.price_number = cast(products.price, 'number');
metric product_rank_by_price = rank(order: products | max(products.price_number) | desc());
explore {
  dimensions {
    product_id: products.id,
    product_name: products.name,
    created_at: products.created_at,
    product_price: products.price_number,
  }
  measures {
    product_rank_by_price: product_rank_by_price,
  }
  filter {
    product_rank_by_price < 11,
  }
  sorts {
    product_price DESC,
  }
}
```

## Learning points
* We want to compare prices as numbers, but products.price is of type "text" => need to cast it to "number" first

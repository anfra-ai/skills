## Dataset fields
```yaml
dimensions:
- field: created_at

- field: avg_age

```

## AQL
```aql
explore {
  dimensions {
    created_at: db_users.created_at | month(),
  }
  measures {
    avg_age_previous_year: db_users.avg_age | relative_period(db_users.created_at, interval(-1 year)),
  }
  filters {
    db_users.created_at is @(beginning - this month),
  }
  sorts {
    created_at ASC,
  }
}
```

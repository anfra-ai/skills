## Dataset fields
```yaml
dimensions:
- field: sign_up_at
  model: users
- field: id
  model: users
- field: first_name
  model: users
- field: last_name
  model: users
- field: name
  model: city
measures:
- field: avg_age
  model: ''
```

## AQL
```aql
dimension users.full_name_and_city = concat(users.first_name, ' ', users.last_name, ' - ', city.name);
explore {
  dimensions {
    id: users.id,
    full_name_and_city: users.full_name_and_city,
  }
  sorts {
    id ASC,
  }
}
```

## Learning points
* Calculations (`concat` in this case) must be refactored into a new dimension on the source model.

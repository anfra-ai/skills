## Dataset fields

## AQL
```aql
explore {
  measures {
    avg_age_last_month: db_users.avg_age,
    avg_age_last_year_same_month: db_users.avg_age | relative_period(db_users.created_at, interval(-1 year)),
  }
  filters {
    db_users.created_at is @(last month),
  }
  sorts {
    avg_age_last_month ASC,
  }
}
```

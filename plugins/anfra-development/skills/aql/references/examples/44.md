## AQL
```aql
metric product_rank_by_price_in_each_city = rank(
  order: product | max(product.price) | desc(),
  partition: public_cities.id,
);

explore {
  dimensions {
    city_id: public_cities.id,
    city_name: public_cities.name,
    product_id: product.id,
    product_name: product.name,
    product_price: product.price,
  }
  filters {
    product_rank_by_price_in_each_city == 1,
  }
  sorts {
    city_id ASC,
    product_id ASC,
  }
}
```

## Learning points
* Using `rank()` with `partition` for "top in category" questions

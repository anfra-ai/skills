## Dataset fields
```yaml
dimensions:
- field: id
  model: orders
- field: discount
  model: orders
measures:
- field: value
  model: orders
```

## AQL
```aql
metric revenue =
  unique(orders.id)
  | select(
     // need to turn discount (per order) into a metric to do metric subtraction
    revenue_per_order: orders.value - (orders | sum(orders.discount)),
  )
  | sum(revenue_per_order);

explore {
  dimensions {
    id: orders.id,
  }
  measures {
    revenue: revenue,
  }
  sorts {
    id ASC,
  }
}
```

## Learning points
* Dimensions (e.g. `orders.discount`) and metrics/measures (e.g. `orders.value`) cannot be mixed directly in an expression. Need to aggregate or group the dimensions.

## Dataset fields
```yaml
dimensions:
- field: id
  model: orders
- field: discount
  model: orders
- field: order_id
  model: items
- field: price
  model: items
measures: []
```

## AQL
```aql
metric total_price = items | sum(items.price);
metric total_discount = orders | sum(orders.discount);
metric revenue =
  unique(orders.id)
  | select(
    revenue_per_order: total_price - total_discount,
  )
  | sum(revenue_per_order);

explore {
  dimensions {
    id: orders.id,
  }
  measures {
    revenue: revenue,
  }
  sorts {
    id ASC,
  }
}
```

## Learning points
* Dimensions (e.g. `orders.discount`) and metrics/measures (e.g. `sum(items.price)`) cannot be mixed directly in an expression. Need to aggregate or group the dimensions.

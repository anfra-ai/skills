## AQL
```aql
metric user_count = users | count(users.id);
metric country_rank_by_user_count = rank(order: user_count | desc());

explore {
  dimensions {
    country_code: countries.code,
    country_name: countries.name,
  }
  measures {
    user_count: user_count,
    country_rank_by_user_count: country_rank_by_user_count,
  }
  filter {
    country_rank_by_user_count < 11,
  }
  sorts {
    user_count DESC,
  }
}
```

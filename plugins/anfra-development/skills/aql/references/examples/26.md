## AQL
```aql
dimension orders.revenue = orders.value - orders.discount;
metric revenue = orders | sum(orders.revenue);

explore {
  dimensions {
    time: orders.created_at | month(),
  }
  measures {
    revenue: revenue,
  }
  sorts {
    time ASC,
  }
}
```

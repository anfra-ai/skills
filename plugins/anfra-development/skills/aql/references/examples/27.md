## Dataset fields
```yaml
dimensions:
- field: id
  model: orders
- field: discount
  model: orders
- field: order_id
  model: items
- field: price
  model: items
measures: []
```

## AQL
```aql
// put dimension on items because it is on the many side (an order has many items, but an item has only 1 order = 1 discount)
dimension items.revenue = items.price - orders.discount;

explore {
  dimensions {
    id: orders.id,
  }
  measures {
    revenue: items | sum(items.revenue),
  }
  sorts {
    id ASC,
  }
}
```

## Bad AQL
```aql
explore {
  dimensions {
    order_id: orders.id,
    user_id: users.id,
    user_signed_up_at: users.signed_up_at
  }
  filters {
    users.id == orders.user_id,
    users.signed_up_at in @(2024),
  }
  sorts {
    order_id ASC,
  }
}
```

## Why it's bad
* `users.id == orders.user_id` is **bad**. NEVER try to implement joins in AQL.

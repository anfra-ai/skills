## Bad AQL
```aql
metric product_rank_by_price = rank(order: product.price | desc());

explore {
  dimensions {
    product_id: product.id,
    product_name: product.name,
  }
  measures {
    product_price: max(product.price),
    product_rank_by_price: product_rank_by_price,
  }
  having {
    product_rank_by_price < 4,
  }
  sorts {
    city_id ASC,
    product_id ASC,
  }
}
```

## Why it's bad
* `rank()` expects a **metric** in `order:` but `product.price` is a **dimension**.

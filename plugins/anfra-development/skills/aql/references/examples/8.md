## AQL
```aql
dimension products.order_count = orders | count(orders.id) | dimensionalize(products.id);
explore {
  dimensions {
    product_id: products.id,
    product_name: products.name,
    created_at: products.created_at,
  }
  sorts {
    created_at DESC,
    product_name ASC,
  }
  filters {
    products.order_count >= 20,
  }
  limit: 10
}
```

## Dataset fields
```yaml
dimensions:
- field: id
  model: orders
- field: discount
  model: orders
- field: name
  model: city
measures:
- field: value
  model: orders
```

## AQL
```aql
metric avg_order_value_per_city =
  unique(city.id)
  | select(value: orders.value)
  | avg(value);

explore {
  measures {
    avg_order_value_per_city: avg_order_value_per_city,
  }
  sorts {
    avg_order_value_per_city DESC,
  }
}
```

## Learning points
* `orders.value` is a measure and we need to calculate average on that measure => Need to do a nested aggregation

## AQL
```aql
metric previous_month_user_count = user_count | previous(order: x_users.created_at | month());
metric user_count_growth_rate = user_count * 100.0 / previous_month_user_count;
explore {
  dimensions {
    sign_up_month: x_users.created_at | month(),
  }
  measures {
    user_count: user_count,
    previous_month_user_count: previous_month_user_count,
    user_count_growth_rate: user_count_growth_rate,
  }
  sorts {
    sign_up_month ASC,
  }
}
```
